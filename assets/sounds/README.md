# Custom Sounds

Put your audio files in this folder with these exact names:

- `hammer-hit.mp3` - hammer hit sound.
- `hammer-miss.mp3` - miss sound.
- `character-normal.mp3` - normal character reaction.
- `character-bonus.mp3` - bonus character reaction.
- `character-fast.mp3` - fast character reaction.
- `target-pop.mp3` - target appears from the hatch.
- `round-start.mp3` - round start sound.
- `game-over.mp3` - game over sound.
- `record.mp3` - new record sound.
- `menu-music.mp3` - looping music on the main/menu screen.
- `game-music.mp3` - looping music during the round.
- `end-music.mp3` - looping music on the final score screen.

Use short compressed files for Telegram. MP3 is the default supported format in the code.
